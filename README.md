# icon-demo

A React icon component library powered by [Lsicon](https://github.com/wisdesignsystem/lsicon-figma).

## Preview

[https://lsicon.com?package=icon-demo](https://lsicon.com?package=icon-demo)

## Installation

### npm

```bash
npm i icon-demo
```

### yarn

```bash
yarn add icon-demo
```

### pnpm

```bash
pnpm i icon-demo
```

## Usage

```tsx
import { IconAIcon } from "icon-demo";

function App() {
  return <IconAIcon />;
}
```
